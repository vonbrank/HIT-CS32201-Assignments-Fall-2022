# 哈工大《计算机组织与体系结构》实验 - 2022年秋

## 实验内容

### Assignment 01

实现一个 ALU

### Assignment 02

给出一个指令系统（大概是 `MIPS` 的一个子集），实现一个支持此指令系统的处理器，仅需要顺序实现。

### Assignment 03

将 `Assignment 02` 中的顺序实现修改为流水线。

此项为组队实验，这里只给出通过插入气泡和暂停实现的流水线。

### Assignment 04

实现一个指令 Cache，具体来说需要实现 CPU 与 Cache 之间的 Sram 协议和 Cache 与主存之间的 AXI 协议
